# PyMKMKit

[![CI](../../actions/workflows/ci.yml/badge.svg)](../../actions/workflows/ci.yml)

Python-based MicroKinetic Modelling Kit

## Installing

Clone this repository and install using. It is recommended to use PyMKMKit
in a virtual environment.

```bash
pip install .
```

To test the kit:

```bash
python -m pytest -s
```
